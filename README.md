# 은퇴자금 시뮬레이터

단일 HTML 파일(`index.html`)로 동작하는 한국형 은퇴자금 시뮬레이터입니다. 빌드·서버 없이 브라우저에서 바로 열면 됩니다.

## 주요 기능 (v3.4)

- **단일 시뮬레이션**: 시기별(적립기/위험기/후기) 수익률, 포트폴리오 프리셋 16종, 글라이드 패스(Bond Tent)
  - 글라이드 패스 = 전환 시작 연도부터 수익률·변동성을 적립기→위험기 수준으로 점진 블렌딩(선형/ease-in/ease-out), 전환 완료 후~은퇴 전엔 위험기 수준 유지. 은퇴 직후 폭락(시퀀스 리스크)에 대비해 미리 보수화하는 전략의 근사 모델
- **Monte Carlo**: 정규분포 수익률 경로 시뮬레이션, 5/25/50/75/95% 밴드, 성공률 + 말년 자산>0 비율
- **위기 시나리오**: 역사적 위기 6종(IMF/닷컴/GFC/코로나/긴축/잃어버린 10년) + 사용자 정의 — 단일·MC 모두 결합 가능
- **현금흐름 모델**: Trinity 4% 인출(인플레 조정) + 국민연금(개시연령 입력, 인플레 연동) + 비과세연금/KB보험(개시연령 기준, 명목 고정) + 주택연금(HF공사 종신정액형 테이블 보간)
- **세금·건보료**: 인출 실효세율 + 지역건보료(인플레 연동) 차감 — 표의 총 월수령은 세후 기준
- **변동성 드래그 보정**: 단일 시뮬을 기하평균 근사(μ−σ²/2)로 계산해 MC 중앙값과 비교 가능
- **환헷지 토글**: 변동성 복원 + 헷지 비용(%/년) 수익률 차감
- **기준선 비교**: 현재 결과를 📌 기준선으로 저장 후 다른 설정과 차트 오버레이 비교
- **입력값 저장**: 변경 시 localStorage 자동 저장(브라우저 로컬), JSON 내보내기/가져오기, 초기화

## ⚠️ 보안 주의

- 이 앱은 100% 클라이언트 사이드입니다. **잠금화면은 어깨너머 시선을 막는 가림막일 뿐 보안 장치가 아닙니다** (소스 보기·devtools로 우회 가능).
- 따라서 **실제 개인 재무 수치를 `index.html` 기본값에 하드코딩해 커밋하지 마세요.** 코드의 기본값은 일반적인 예시 숫자입니다.
- 본인 수치는 화면에서 직접 입력하세요 — localStorage(해당 브라우저에만)와 JSON 파일(직접 보관)로 관리됩니다. JSON 파일도 저장소에 커밋하지 마세요.
